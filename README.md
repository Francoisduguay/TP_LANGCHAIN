# TP_LANGCHAIN
---

## 1. Make sure you install the dependencies:
From project root, run:
- ```pip install -r requirements.txt```

## 2. Create and populate db
From project root, run:
- ```python init_db.py```

## 3. To chat in terminal
- ```python main.py```

## 4. To start the streamlit interface
- ```streamlit run app.py```
